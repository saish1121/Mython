# Solution..............

1.  Writing the Python source code (.py file)
    This file contains human-readable code (high-level language).
    Computers cannot understand this directly.

2.  Compilation (Source code â†’ Bytecode)

    a) Syntax Checking

    (Missing bracket â†’ program stops here.)

    b) Convert to Bytecode
    If no syntax error:
    Python compiles the code into bytecode
    Bytecode is:
    Low-level
    Platform-independent
    Faster than source code

    (This step is done by Python Compiler)

3.  Python Virtual Machine (PVM) Execution

    The Python Virtual Machine (PVM) takes the bytecode
    PVM reads bytecode line by line
    Converts it into machine-level instructions
    Executes it on the CPU

    (This is where the program actually runs and produces output)

4.  Output is produced 

    Example:

        print("Hello World")


    Output:

        Hello World
